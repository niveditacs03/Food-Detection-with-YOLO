# YOLOv8 Object Detection with Pre-processing

This repository delivers a streamlined, production-aligned pipeline combining image pre-processing with YOLOv8 for high-fidelity object detection. The notebook operationalizes data readiness, model execution, and result visualization in a single, modular workflow.

## Key Capabilities

- End-to-end pre-processing (resize, normalize, enhance).

- YOLOv12 training and inference workflow.

- Modular, reusable code architecture.

- Annotated predictions and performance insights.
